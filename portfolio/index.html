Maze Game

This is my first game. In 8th grade, I was interested in maze generation. By myself, I developed two algorithms for generating mazes, which I later learned were called Kruskal's algorithm and Prim's algorithm. Learning the names of the algorithms that I had "invented" was a little saddening, but exciting at the same time. This is actually how I was introduced to graph theory.

The main page is "popup.html" rather than "index.html", because this was origonally made as a chrome extension. At this time, the extension is not available for download.
